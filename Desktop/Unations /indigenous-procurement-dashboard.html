API Integration Documentation

Authentication

All API requests require authentication using OAuth 2.0. Include your access token in the Authorization header.

Authorization: Bearer YOUR_ACCESS_TOKEN

Endpoints

GET /api/v1/departments

Retrieve all department procurement data

{ "departments": [ { "id": "ISC", "name": "Indigenous Services Canada", "compliance_rate": 18.4, "target": 5.0, "total_contracts": 1247, "indigenous_contracts": 229, "total_value": 458000000, "indigenous_value": 84272000, "verification_status": "verified", "last_audit": "2024-10-15" } ] }
GET /api/v1/suppliers/{supplier_id}

Get detailed supplier verification information

{ "supplier": { "id": "S123456", "name": "Northern Tech Solutions Inc.", "business_number": "123456789RC0001", "indigenous_ownership": 51, "verification_status": "verified", "certifications": ["CCAB", "ISO-9001"], "active_contracts": 3, "total_contract_value": 3750000 } }

Submit a new supplier for verification
